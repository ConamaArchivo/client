<h2>To run it locally:</h2>

1.  Clone this repo
```bash
git clone git@github.com:ConamaArchivo/client.git
```

2. Change directory
```bash
cd client
```

3. Install dependencies
```bash
npm install
```

3. Run as development
```bash
npm start
```
